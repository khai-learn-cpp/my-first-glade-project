# Khai's first Gtk::Builder project

## Prerequisites

* gtkmm-3.0

* GCC > 5.0

* Bash (Shell)

## Building

### Environment variables

* `CXX`: Compiler command

* `PKG_CONFIG_PATH`: Read GTK+'s docs for more information

### Command line

**Build**

```bash
cd repo-root
sh build
```

**Run**

```bash
repo-root/main
```
