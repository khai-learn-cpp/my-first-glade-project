## Prerequisites

* gtkmm-3.0

* GCC > 5.0

* Bash (Shell)

* NodeJS and npm

## Development

### Environment variables

* `CXX`: Compiler command

* `PKG_CONFIG_PATH`: Read GTK+'s docs for more information

* `SKIP_QUICK_TEST` (optional): Either be `TRUE` or `FALSE`

### Test

```bash
npm test
```

### Build

```bash
npm run build
```

### Run

```bash
npm start
```
